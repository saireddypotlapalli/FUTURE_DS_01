# 📊 Business Sales Performance Analytics

## 📌 Project Overview
This project analyzes real-world business sales data to identify revenue trends, profitability, and growth opportunities.  
An interactive dashboard is built to support data-driven business decision making.

---

## 🎯 Objectives
- Identify top revenue-generating products
- Analyze sales trends over time
- Detect profitable regions and categories
- Provide actionable business recommendations
- Build a client-ready analytics dashboard

---

## 🛠 Tools & Technologies
- Python
- Pandas
- Streamlit
- Plotly
- NumPy
- Scikit-learn

---

## 📊 Dashboard Features
- Executive KPI metrics (Revenue, Profit, Margin, Orders)
- Sales trend analysis
- Category profitability
- Regional performance comparison
- Top & bottom product analysis
- Business insights & recommendations
- Interactive filters
- Client-ready layout

---

## 📈 Business Impact
- Helps identify high-performing regions and products
- Supports strategic business decisions
- Detects growth trends and risks
- Improves profitability planning

---

## ▶️ How to Run the Project

```bash
pip install -r requirements.txt
streamlit run dashboard/dashboard.py
```

Then open:
```
http://localhost:8501
```
